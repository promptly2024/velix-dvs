## Writing Guidelines (for each article)
- Prerequisites: What the user needs (IDs, account access, documents).
- Exact Steps: Numbered instructions; include official link buttons and field names.
- Screenshots: Highlight critical UI elements (blur sensitive info).
- Time & Outcome: Typical timelines and what confirmation looks like.
- Safety Tips: What not to do; avoiding scams; post-action security (2FA, passwords).
- Escalation: Where to complain/escalate if ignored (with links).
